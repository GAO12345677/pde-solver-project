本附录仅保留系统中最能体现论文核心设计的代码节选，主要包括特征提取、算法选择、数值求解和自动求解流程四个部分。

## A.1 特征提取模块

该模块将方程维度、线性特征、边界条件、问题规模以及求解需求转化为统一数值向量，作为算法选择模型的输入。

```python
class PhysicsFeatureExtractor:
    @staticmethod
    def extract_from_params(params):
        boundary_map = {"dirichlet": 0.0, "neumann": 0.5, "mixed": 1.0}
        return np.array([
            int(params["dimension"]),
            int(params.get("linearity", 0)),
            int(params.get("stationarity", 0)),
            boundary_map[params.get("boundary_condition", "dirichlet")],
            float(params["problem_size"]),
        ], dtype=float)

    @staticmethod
    def normalize(vec):
        return np.array([
            (vec[0] - 1.0) / 2.0,
            vec[1],
            vec[2],
            vec[3],
            min(max((vec[4] - 1e3) / (1e7 - 1e3), 0.0), 1.0),
        ], dtype=float)


def build_feature_vector(physics, hardware, domain):
    return np.concatenate([physics, hardware, domain], axis=0).astype(np.float32)
```

## A.2 算法自适应选择模块

该模块根据物理特征、硬件特征和领域需求，对有限差分法、有限元法和谱方法进行综合评分，并输出推荐算法。

```python
def evaluate_algorithm(algorithm_key, physics, hardware, domain):
    dim, nonlinear, unsteady, _, size = physics
    accuracy_need, realtime_need, resource_budget = domain
    parallel_level = hardware[-1]

    base = {
        "fdm": {"acc": 0.55, "conv": 0.75, "res": 0.85},
        "fem": {"acc": 0.75, "conv": 0.70, "res": 0.60},
        "spectral": {"acc": 0.92, "conv": 0.65, "res": 0.55},
    }[algorithm_key]

    acc = base["acc"]
    conv = base["conv"] - 0.10 * nonlinear - 0.05 * unsteady
    res = base["res"] - 0.25 * size + 0.15 * parallel_level

    if algorithm_key == "spectral":
        acc += 0.10 * accuracy_need - 0.15 * dim
    elif algorithm_key == "fem":
        acc += 0.10 * dim + 0.05 * nonlinear
    else:
        res += 0.05 * (1 - dim)

    w_acc = 0.45 + 0.15 * accuracy_need
    w_conv = 0.25 + 0.10 * realtime_need
    w_res = 0.30 + 0.20 * (1 - resource_budget)
    total_weight = w_acc + w_conv + w_res

    score = (
        w_acc * np.clip(acc, 0, 1)
        + w_conv * np.clip(conv, 0, 1)
        + w_res * np.clip(res, 0, 1)
    ) / total_weight
    return float(score)


def select_algorithm(physics, hardware, domain):
    candidates = ["fdm", "fem", "spectral"]
    scores = {
        key: evaluate_algorithm(key, physics, hardware, domain)
        for key in candidates
    }
    return max(scores, key=scores.get), scores
```

## A.3 数值求解核心模块

该模块以一维热传导方程为例，使用有限差分方法离散空间二阶导数，并通过时间积分器获得数值解。

```python
class FiniteDifferenceSolver:
    algorithm_key = "fdm"

    def solve(self, k, L, nx, t_span, initial, left_bc=0.0, right_bc=0.0):
        x = np.linspace(0.0, L, nx)
        dx = x[1] - x[0]
        u0 = np.asarray(initial(x), dtype=float)
        u0[0], u0[-1] = left_bc, right_bc

        def laplacian(u):
            d2 = np.zeros_like(u)
            d2[1:-1] = (u[2:] - 2.0 * u[1:-1] + u[:-2]) / (dx * dx)
            return d2

        def rhs(t, u):
            u[0], u[-1] = left_bc, right_bc
            du = k * laplacian(u)
            du[0], du[-1] = 0.0, 0.0
            return du

        start = time.perf_counter()
        sol = solve_ivp(rhs, t_span, u0, rtol=1e-6, atol=1e-9)
        elapsed = time.perf_counter() - start

        u_end = sol.y[:, -1]
        u_end[0], u_end[-1] = left_bc, right_bc
        return u_end, {"algorithm": self.algorithm_key, "elapsed_s": elapsed}
```

## A.4 自动求解流程模块

该模块串联自然语言解析、特征提取、算法推荐和数值求解，实现从题目输入到结果输出的闭环过程。

```python
async def auto_solve(question, parser):
    parsed = await parser.parse(question)
    physics_params = parsed["physics_params"]
    domain_demand = parsed["domain_demand"]

    physics = PhysicsFeatureExtractor.extract_from_params(physics_params)
    physics = PhysicsFeatureExtractor.normalize(physics)

    hardware = HardwareFeatureExtractor.normalize(HardwareFeatureExtractor.extract())
    domain = DomainFeatureExtractor.normalize(domain_demand)

    algorithm_key, scores = select_algorithm(physics, hardware, domain)

    solver = get_solver(algorithm_key)
    solution, solve_info = solver.solve(
        k=physics_params.get("k", 1.0),
        L=physics_params.get("L", 1.0),
        nx=physics_params.get("problem_size", 101),
        t_span=(0.0, physics_params.get("t1", 0.1)),
        initial=lambda x: np.sin(np.pi * x),
    )

    return {
        "parsed": parsed,
        "selected_algorithm": algorithm_key,
        "algorithm_scores": scores,
        "solution": solution.tolist(),
        "solve_info": solve_info,
    }
```

以上代码展示了系统的核心闭环：首先将物理问题转化为可计算特征，然后根据多指标评分选择合适算法，最后调用对应数值求解器并返回结果。
